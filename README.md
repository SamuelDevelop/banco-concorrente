# Banco Concorrente

A ideia aqui é fazer uma simulação de um banco, que controla dados bancários e gerencias transferências. Meu objetivo é atrás dessa ideia explorar programação Multi-Thread, com ações paralelas sendo executadas ao mesmo tempo. Além de reforçar meus conhecimentos na linguagem Java.
